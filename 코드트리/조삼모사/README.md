n 개 중 n/2 개를 뽑는 조합(백트래킹) 문제

해설

- 해설 - n 크기를 가지는 Boolean 배열에서 5개를 True 로 변경하는 방식
- 나 - 빈 배열에 5개의 일을 집어넣는 방식
    - 오전 오후의 업무 강도를 찾아내는 과정에 차이가 발생
        - 난 정해진 갯수(n//2) 만큼만 돌면서 찾아내지만, 해설은 n 만큼 돌면서 찾아내기에 시간에서 차이가 남
